# River Data Structure

Learn about how river data is organized in our API.

## River Object Structure
```json
{
  "id": "river-id",
  "title": "River Name",
  "description": "River description",
  "files": []
}
```

## Data Types
Each river includes:
1. Basic Information
2. Snow Cover Data
3. Dynamic Measurements
4. Historical Records
